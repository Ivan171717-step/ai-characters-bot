# AI Characters Telegram Bot

Telegram AI-бот с двумя персонажами:

- 👩 Анастейша — поддерживающее и мягкое общение
- 😏 Джейсон — сарказм, юмор и подколы

## Возможности

- выбор персонажа
- AI-ответы через OpenAI
- сохранение истории диалогов
- SQLite база данных
- подготовка к Railway
